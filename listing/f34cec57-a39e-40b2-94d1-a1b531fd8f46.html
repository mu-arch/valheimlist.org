About the Server

We are a community that hosts servers, and have added Valheim to out list! We are making this Valheim server a Season Based Server, meaning resets will happen, but will be announced in advance rather than randomly. Server is PvE. This Valheim server is a "Modded, Semi-Vanilla" server that uses mods such as "server character" and "time on screen." How do I join? To join the server, join the Discord and accept the rules. Then role react for the Valheim channels and follow the instructions in #v-info. It will tell you how to get the manager and auto install mods to easily join with little work. Most players get this done in about 3 minutes. What is a Season Based Server? What this means is we will reset the server after several months and make the world available for download. The resets have not been decided yet but a poll will be run when we have enough members to make sure everyone's input is heard. Your adventure beings here!
